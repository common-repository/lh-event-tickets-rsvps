Improves the display of Modern Trtibes Event Tickets RSVP's, dramatically

== Description ==

This plugin improves how Modern Tirbies Event Tickets plugin, by displaying RSVP's without the need for any paid add ons, and in a better way.

Improvements include, 

* Buddypress Avatars (if buddypress is loaded).
*Gravatar fallback (in the absence of a buddypress avatar or if buddypress is not actiavted).
* Displaying the name of the person who has RSVPed

More Features to come

== Installation ==

1. Ensure the Event Ticketsd plugin is installed and activated
1. Upload the `lh-event-tickets-rsvps` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. That is it!
